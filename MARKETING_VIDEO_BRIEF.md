## Technical Production Notes

### Screen Recording Requirements

**Web App Recordings:**
1. Landing page scroll (1920x1080, 60fps)
2. Job posting flow - complete 3 steps (capture browser at 1440x900)
3. Contractor dashboard - job browsing with filters (1920x1080)
4. Bid comparison interface (1920x1080)
5. Mapbox aerial imagery interactions (zoom, pan, marker drop)

**Mobile App Recordings:**
- Use iOS Simulator recording for consistency
- Portrait orientation: 390x844 (iPhone 14 Pro)
- Record: Job feed scrolling, job detail view, bid submission
- Enable demo mode (hide status bar elements)

**Code/Tech Visualizations:**
- VS Code with Aerial Platform project open
- Database schema: Use dbdiagram.io or similar
- Tech stack icons: Use official brand assets
- Metrics dashboard: After Effects animation or Figma prototype

### Color Palette
- **Primary:** #2563eb (Blue) - trust, tech
- **Secondary:** #10b981 (Green) - growth, eco-friendly
- **Accent:** #f59e0b (Amber) - energy, action
- **Background:** #ffffff / #f9fafb
- **Text:** #111827

### Typography
- **Headings:** Inter Bold / SF Pro Display Bold
- **Body:** Inter Regular / SF Pro Text
- **Code:** JetBrains Mono

### Music Recommendations
- Epidemic Sound: "Tech Innovation" or "Digital Future"
- Artlist: "Corporate Tech Upbeat" category
- Length: 75-90 seconds, building energy, resolution at end
- BPM: 120-130 (energetic but professional)

### Voiceover Direction
- **Gender:** Neutral or male (confident, authoritative)
- **Pace:** Moderate-fast (but clear enunciation)
- **Tone:** Enthusiastic but professional (not salesy)
- **Accent:** Neutral American or British
- **Recording:** Studio quality, minimal reverb
